# Subnetting Manual – Do Zero à Prática

Este projeto tem como objetivo ensinar subnetting de forma manual,
sem uso de calculadoras ou aplicativos, utilizando lógica, matemática
simples e simulações no Cisco Packet Tracer.

## O que você vai aprender
- O que é Network, Broadcast e Hosts
- Como funciona o CIDR (/24, /26, /30)
- Como calcular sub-redes manualmente
- Como aplicar subnetting em redes reais

## Ferramentas utilizadas
- Cisco Packet Tracer
- Markdown
- GitHub
